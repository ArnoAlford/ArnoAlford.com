Network Observability with pfSense

A project to use monitoring, automation and metrics to evaluate and improve my home network on the pfSense platform.

Who Am I Communicating With?

Using ntopng, a package for pfSense, I'm able to grab traffic statistics from my router. From this I can see some really impactful information, such as what IP's I am connecting with, how much of my traffic is going towards each IP, and what ports am I communicating on the most.

What Applications are Doing the Communicating?
One of the most useful metrics I can grab from ntopng is application protocol traffic. Here I can see exactly what applications are talking on my network, how much bandwidth they've used and how long they've been up. This opens up some interesting questions such as "Why am I talking with Facebook if I don't have an account with them? Where is that traffic originating from?"
Increasing Performance Through Caching

By using squid as a transparent proxy on the network we can analyze and cache frequently visited pages. What does this mean? It means that the assets for frequently visited websites are loaded faster because the squid proxy saves them on the server. This means reduced page load times and reduced bandwidth usage.

Caching Encrypted Traffic
If left default, the only traffic my proxy could read and cache would be HTTP; however most taffic today is over encrypted HTTPS. To get around this I had to make a CA (Certificate Authority) to issue certificates that allowed me to splice HTTPS packets. This means that all web traffic, even the encrypted, can now be cached.
Automating Backups

One of the features not natively supported in basic pfSense is automatic backups, so I wrote this script to automatically create a backup once a day. The script is run on a remote server, and it pulls the backup from the router using the login form and a token. In addition, I have the script delete any backups older than 10 days every time it runs.

Automating Updates
Another important feature for me was updating the router. I always forget to install updates when they come out, so I wrote a script to automatically check and install any updates that are avaliable.
